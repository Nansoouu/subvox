# Subvox Community

Welcome to the Subvox community. This project is 100% open-source and community-driven. Here's how everything works.

## Values

- **Transparency** — All pool logic is public code in this repo
- **Decentralization** — No single point of failure, no VC control
- **Fairness** — Contributors get rewarded, users get a free tier

## How to participate

| Activity | How |
|----------|-----|
| **Report a bug** | Open an issue on [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) |
| **Request a feature** | Open an issue with the `enhancement` label |
| **Submit code** | Fork, branch, PR. See `CONTRIBUTING.md` in subvox-pipeline |
| **Translate the project** | Add/improve language support |
| **Donate GPU compute** | DM on Discord (link coming) |

## Communication

- **GitHub Issues** — Bugs, features, questions
- **Discord** (soon) — Real-time chat
- **X/Twitter** — [@subvoxapp](https://x.com/subvoxapp)

## Token system

Subvox uses **SUBTEST** (Solana devnet) to reward contributors and power the community economy.

- 1 SUBTEST = 1 community credit
- Contributors earn tokens for merged PRs
- Users in the community pool get free transcription credits

See the full tokenomics in `rewards/tokenomics.md`.
