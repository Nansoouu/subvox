## Que es Subvox?

Subvox es un **motor de subtitulos open-source** para la comunidad. Pega una URL de video, elige un idioma, y obten un video subtitulado en minutos.

**Mas de 1800 plataformas soportadas** via yt-dlp.

## Por que Subvox?

La traduccion de video afecta a **miles de millones de personas**. Un tutorial coreano, una conferencia alemana, una clase arabe, un vlog espanol. El idioma nunca deberia ser una barrera.

**El problema:** Las herramientas existentes (Veed, Kapwing, Descript, Opus Clip) limitan el tiempo, las exportaciones y te encierran en suscripciones caras.

**La solucion:** Una comunidad descentralizada donde todos ganan.

| Quien | Que obtienen |
|-------|-------------|
| **Usuarios intensivos** | Traduccion ilimitada a bajo precio. 21 idiomas. |
| **Duenos de claves Groq** | Compartan claves no usadas, ganen 55% por traduccion. |
| **Tenentes de tokens** | Tengan 500K+ SUBVOX, ganen recompensas diarias. |
| **Contribuidores** | Reporten bugs, escriban codigo, traduzcan. Gane SUBVOX. |
| **Todos** | Sistema transparente y auditable. Garantizamos que todos reciben pago. |
