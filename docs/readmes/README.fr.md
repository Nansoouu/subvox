# Subvox — README Francais

Lisez le README principal en anglais pour la version complete. Voici les informations cles en francais.

## Qu'est-ce que Subvox ?

Subvox est un **moteur de sous-titrage open-source** pour la communaute. Collez une URL video, choisissez une langue, obtenez une video sous-titree en minutes.

**Plus de 1800 plateformes supportees** — X/Twitter, TikTok, YouTube, Instagram, Vimeo, Dailymotion, et des milliers d'autres via yt-dlp.

## Comment ca marche

1. **Collez une URL** — depuis n'importe quelle plateforme parmi 1800+
2. **Choisissez une langue** — 21 langues disponibles
3. **Attendez 1-10 minutes** — le pipeline s'occupe de tout
4. **Obtenez votre video** — sous-titres incrustes + SRT/VTT telechargeables

## Comment gagner

Subvox est concu pour que **tout le monde puisse gagner** en contribuant a l'ecosysteme, sans savoir coder.

**1. Partagez votre cle Groq** — Pretez vos credits Groq a la pool communautaire et gagnez des SUBTEST proportionnellement a l'utilisation.

**2. Partagez vos cookies** — Certains sites (YouTube...) necessitent des cookies. Partagez-les de maniere securisee et gagnez des rewards.

**3. Detenez des SUBTEST** — Les holders recoivent une part des revenus de la plateforme.
