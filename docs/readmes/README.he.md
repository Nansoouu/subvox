## למה Subvox?

תרגום וידאו משפיע על **מיליארדי אנשים**. מדריך קוריאני, כנס גרמני, הרצאה ערבית, בלוג ספרדי. שפה לעולם לא צריכה להיות מחסום.

**הבעיה:** כלים קיימים (Veed, Kapwing, Descript, Opus Clip) מגבילים אותך לכמה דקות, מגבילים את הייצוא שלך, ונועלים אותך למנויים יקרים. הם גובים הון עבור מה שצריך להיות נגיש.

**הפתרון:** קהילה מבוזרת שבה כולם מרוויחים.

| מי | מה הם מקבלים |
|-----|---------------|
| **משתמשים כבדים** | תרגמו סרטונים ללא הגבלה בשבריר ממחירי המתחרים. ללא מגבלות זמן. 21 שפות. |
| **מחזיקי מפתח Groq** | שתפו מפתחות שאינכם משתמשים בהם והרוויחו חלק ממאגר הספקים של 60%. הכנסה פסיבית. |
| **מחזיקי אסימונים** | הפקדו 500K+ SUBVOX והרוויחו תגמולים יומיים ממאגר המחזיקים של 20%. |
| **מתקני באגים ותורמים** | דווחו על באגים, כתבו קוד, תרגמו שפות. הרוויחו SUBVOX בעל ערך כספי אמיתי. |
| **כולם** | שקוף, ניתן לביקורת, על-גבי השרשרת. אנו מאשרים שכולם מקבלים תשלום והמערכת עובדת. |

ללא מנוי. ללא עמלות נסתרות. ללא אג'נדה תאגידית. נבנה על ידי הקהילה, למען הקהילה.

## איך הצינור עובד

כל תרגום עובר 5 שלבים. הנה איך **הקהילה** מפעילה כל אחד מהם.

> 💡 **רוצים לראות איך זה עובד מתחת למכסה המנוע?** הצינור הוא קוד פתוח מלא בכתובת [github.com/Nansoouu/subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline). אתם יכולים להריץ אותו בעצמכם, לבדוק כל שורת קוד, ולתרום.

### 1. הורדה
כתובת ה-URL של הסרטון שלכם מורדת באמצעות yt-dlp. חברי קהילה יכולים לשתף עוגיות מוצפנות כדי לפתוח פלטפורמות מוגבלות (יוטיוב וכו').

**אתם מרוויחים:** שתפו את העוגיות שלכם וקבלו תגמול בכל פעם שהן בשימוש.

### 2. תמלול
דיבור מתומלל לטקסט באמצעות Groq Whisper. **מאגר Groq הקהילתי** מספק נקודות זכות חינמיות לתמלול.

**אתם מרוויחים:** שתפו את מפתח ה-API של Groq שלכם עם המאגר. חלק הספקים (60% מעלות התרגום, המתחלק בין כל השירותים שבשימוש) הולך לספקי המפתחות.

### 3. תרגום
כתוביות מתורגמות לשפת היעד באמצעות DeepSeek (עם גיבוי LLaMA מקומי מתוכנן).

### 4. צריבה
כתוביות נצרבות לתוך הסרטון באמצעות FFmpeg. תמיכה ב-GPU לסרטונים ארוכים מתוכננת.

### 5. מסירה
הסרטון המתורגם שלכם מוכן עם קבצי SRT/VTT להורדה.

## הרוויחו

Subvox מתוכננת כך ש**כל אחד יכול להרוויח** על ידי תרומה למערכת האקולוגית. אין צורך בתכנות.

### 1. שתפו את מפתח Groq שלכם

הוסיפו את מפתח ה-API של Groq שלכם למאגר הקהילתי והרוויחו חלק מ**מאגר הספקים של 60%** בכל פעם שהמפתח שלכם משמש לתמלול.

| קיבולת המפתח שלכם | מה אתם מרוויחים |
|--------------------|---------------|
| 50% אישי / 50% משותף | חצי מהמפתח לעצמכם, חצי למאגר (אתם מרוויחים על שניהם) |
| מגבלה יומית של 30 דקות | עד 30 דק'/יום שמורות לכם, השאר משותף |
| ללא הגבלה | גישה אישית מלאה + תגמולי מאגר מלאים על החלק המשותף |

**איך:** חברו את הארנק שלכם באפליקציית האינטרנט (בקרוב), לכו ל-Billing > Groq Pool, הדביקו את המפתח שלכם, הגדירו את היחס האישי שלכם.

### 2. שתפו את העוגיות שלכם

חלק מהפלטפורמות (יוטיוב וכו') דורשות עוגיות אימות להורדת סרטונים. שתפו את העוגיות המוצפנות שלכם והרוויחו SUBVOX בכל פעם שהן בשימוש.

**איך:** העוגיות שלכם מוצפנות באמצעות Fernet (AES) ומאוחסנות בצורה מאובטחת. אתם יכולים לבטל גישה בכל עת.

### 3. החזיקו אסימוני SUBVOX

החזיקו אסימוני SUBVOX כדי להרוויח חלק מ**מאגר המחזיקים של 20%**, המחולק מדי יום.

| דרגה | יתרה מינימלית | משקל תגמול | הטבות |
|------|----------------|---------------|----------|
| **Decouverte** | 0 SUBVOX | ללא תגמולים | חינם לסרטונים < 2 דק', סימן מים, 3/יום |
| **Passion** | 500,000 SUBVOX | משקל 1x | משך בלתי מוגבל, ללא סימן מים, ייצוא SRT |
| **Builder** | 1,000,000 SUBVOX + מפתח ספק | משקל 2x | תגמולים כפולים, בונוס על הכנסות מפתח הספק שלכם |

**איך זה עובד:** כל 24 שעות, מאגר המחזיקים מתחלק באופן יחסי. אם אתם מחזיקים ב-10% מכלל ה-SUBVOX המשוקלל, אתם מקבלים 10% מהמאגר של אותו יום.

## אסימון SUBVOX ודרגות

### תמחור

עלויות התרגום מחושבות לפי משך הסרטון. המחירים הם ב-SUBVOX, לשפה.

| משך סרטון | מחיר (SUBVOX) |
|---------------|----------------|
| < 2 דקות | **חינם** (דרגת Decouverte) |
| 2-5 דקות | 500 |
| 5-15 דקות | 1,500 |
| 15-30 דקות | 3,000 |
| 30-60 דקות | 6,000 |
| 60-90 דקות | 10,000 |

**הנחת ריבוי שפות:** 20% הנחה על כל שפה נוספת.

### אסימון

| מאפיין | ערך |
|----------|-------|
| שם אסימון | SUBVOX |
| רשת | Solana (devnet לבדיקות, mainnet ייקבע) |
| כתובת אסימון | בקרוב |
| עשרוניות | 6 |

### חלוקת הכנסות לתרגום

| חלק | מי מקבל | למה |
|-------|------------|----------|
| **60%** | ספקים | מתחלק באופן יחסי בין השירותים שבשימוש: Groq (תמלול), DeepSeek (תרגום), OpenRouter (ראייה), עוגיות (גישה לפלטפורמה) |
| **10%** | פלטפורמה | אירוח, פיתוח, תפעול |
| **20%** | מאגר מחזיקים | מחולק מדי יום לכל מחזיקי אסימוני SUBVOX |
| **5%** | נשרף | אסימונים מוסרים לצמיתות מהמחזור |
| **5%** | עמלות תפעול | עלויות תשתית, ממסר API, תחזוקה |

החלוקה מתבצעת **באופן אוטומטי** כל 24 שעות באמצעות cron job. ללא התערבות ידנית.

**דוגמה:** עבודה המשתמשת ב-Groq + DeepSeek מחלקת את ה-60% באופן שווה (30% + 30%). עבודה המשתמשת בכל ארבעת השירותים מחלקת באופן יחסי (25% + 25% + 5% + 5%).

| **עבודת קהילה** | תרגומים, תיעוד, עיצוב, ניהול |

> **חשוב:** SUBVOX הוא אסימון שירות קהילתי. אין לו ערך כספי מובטח. לעולם אל תשלמו כסף עבור SUBVOX.

## ארכיטקטורה

למפתחים שרוצים לצלול לתוך הקוד:

```
Frontend (Next.js)  -->  Pipeline API (FastAPI)  -->  Celery Worker
                                                         |
                                                         |--> Download (yt-dlp)
                                                         |--> Transcribe (Groq Whisper)
                                                         |--> Translate (DeepSeek + LLaMA)
                                                         |--> Burn subtitles (FFmpeg)
                                                         |--> Deliver
```

| מאגר | נראות | מטרה |
|-----------|-----------|---------|
| [subvox](https://github.com/Nansoouu/subvox) | ציבורי | **המאגר הזה** -- מרכז קהילתי, תיעוד, מאגרים |
| [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) | ציבורי | מנוע צינור (קוד פתוח) |
