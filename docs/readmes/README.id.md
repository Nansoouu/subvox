## Mengapa Subvox?

Penerjemahan video memengaruhi **miliaran orang**. Tutorial Korea, konferensi Jerman, kuliah Arab, vlog Spanyol. Bahasa seharusnya tidak pernah menjadi penghalang.

**Masalahnya:** Alat yang ada (Veed, Kapwing, Descript, Opus Clip) membatasi Anda hanya beberapa menit, membatasi ekspor Anda, dan mengunci Anda dalam langganan mahal. Mereka mematok harga tinggi untuk sesuatu yang seharusnya terjangkau.

**Solusinya:** Komunitas terdesentralisasi di mana semua orang menang.

| Siapa | Apa yang mereka dapatkan |
|-------|--------------------------|
| **Pengguna berat** | Terjemahkan video tanpa batas dengan harga sebagian kecil dari pesaing. Tanpa batas waktu. 21 bahasa. |
| **Pemegang kunci Groq** | Bagikan kunci yang tidak Anda gunakan dan dapatkan bagian dari kumpulan penyedia 60%. Penghasilan pasif. |
| **Pemegang token** | Stake 500K+ SUBVOX dan dapatkan imbalan harian dari kumpulan pemegang 20%. |
| **Perbaiki bug & kontributor** | Laporkan bug, tulis kode, terjemahkan bahasa. Dapatkan SUBVOX yang memiliki nilai uang nyata. |
| **Semua orang** | Transparan, dapat diaudit, on-chain. Kami menjamin semua orang dibayar dan sistem berfungsi. |

Tanpa langganan. Tanpa biaya tersembunyi. Tanpa agenda perusahaan. Dibangun oleh komunitas, untuk komunitas.

## Bagaimana pipeline bekerja

Setiap terjemahan melalui 5 langkah. Berikut adalah bagaimana **komunitas** mendukung masing-masing.

> 💡 **Ingin melihat cara kerjanya di balik layar?** Pipeline sepenuhnya open-source di [github.com/Nansoouu/subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline). Anda dapat menjalankannya sendiri, memeriksa setiap baris kode, dan berkontribusi.

### 1. Unduh
URL video Anda diunduh melalui yt-dlp. Anggota komunitas dapat berbagi cookie terenkripsi untuk membuka platform terbatas (YouTube, dll.).

**Anda mendapat imbalan:** Bagikan cookie Anda dan dapatkan imbalan setiap kali digunakan.

### 2. Transkripsi
Ucapan ditranskripsi menjadi teks melalui Groq Whisper. **Kumpulan Groq komunitas** menyediakan kredit transkripsi gratis.

**Anda mendapat imbalan:** Bagikan kunci API Groq Anda dengan kumpulan. Bagian penyedia (60% dari biaya terjemahan, dibagi di semua layanan yang digunakan) diberikan kepada penyedia kunci.

### 3. Terjemahkan
Teks diterjemahkan ke bahasa target melalui DeepSeek (dengan rencana fallback LLaMA lokal).

### 4. Bakar
Teks dibakar ke dalam video melalui FFmpeg. Dukungan GPU untuk video panjang direncanakan.

### 5. Kirim
Video berteks Anda siap dengan file SRT/VTT yang dapat diunduh.

## Dapatkan Imbalan

Subvox dirancang sehingga **siapa pun dapat mendapat imbalan** dengan berkontribusi pada ekosistem. Tidak perlu coding.

### 1. Bagikan kunci Groq Anda

Tambahkan kunci API Groq Anda ke kumpulan komunitas dan dapatkan bagian dari **kumpulan penyedia 60%** setiap kali kunci Anda digunakan untuk transkripsi.

| Kapasitas kunci Anda | Apa yang Anda dapatkan |
|----------------------|------------------------|
| 50% pribadi / 50% bersama | Setengah kunci untuk Anda sendiri, setengah untuk kumpulan (Anda mendapat imbalan dari keduanya) |
| Batas harian 30 menit | Hingga 30 menit/hari dicadangkan untuk Anda, sisanya dibagikan |
| Tanpa batas | Akses pribadi penuh + imbalan kumpulan penuh pada bagian yang dibagikan |

**Caranya:** Hubungkan dompet Anda di aplikasi web (segera hadir), buka Billing > Groq Pool, tempel kunci Anda, atur rasio pribadi Anda.

### 2. Bagikan cookie Anda

Beberapa platform (YouTube, dll.) memerlukan cookie autentikasi untuk mengunduh video. Bagikan cookie terenkripsi Anda dan dapatkan SUBVOX setiap kali digunakan.

**Caranya:** Cookie Anda dienkripsi dengan Fernet (AES) dan disimpan dengan aman. Anda dapat mencabut akses kapan saja.

### 3. Pegang token SUBVOX

Pegang token SUBVOX untuk mendapatkan bagian dari **kumpulan pemegang 20%**, dibagikan setiap hari.

| Tingkat | Saldo minimum | Bobot imbalan | Manfaat |
|---------|---------------|---------------|---------|
| **Decouverte** | 0 SUBVOX | Tanpa imbalan | Gratis untuk video < 2 menit, watermark, 3/hari |
| **Passion** | 500.000 SUBVOX | Bobot 1x | Durasi tak terbatas, tanpa watermark, ekspor SRT |
| **Builder** | 1.000.000 SUBVOX + kunci penyedia | Bobot 2x | Imbalan 2x, bonus pada pendapatan kunci penyedia Anda |

**Cara kerjanya:** Setiap 24 jam, kumpulan pemegang dibagi secara proporsional. Jika Anda memegang 10% dari semua SUBVOX berbobot, Anda mendapatkan 10% dari kumpulan hari itu.

## Token SUBVOX & tingkatan

### Harga

Biaya terjemahan dihitung berdasarkan durasi video. Harga dalam SUBVOX, per bahasa.

| Durasi video | Harga (SUBVOX) |
|--------------|----------------|
| < 2 menit | **Gratis** (tingkat Decouverte) |
| 2-5 menit | 500 |
| 5-15 menit | 1.500 |
| 15-30 menit | 3.000 |
| 30-60 menit | 6.000 |
| 60-90 menit | 10.000 |

**Diskon multi-bahasa:** Diskon 20% untuk setiap bahasa tambahan.

### Token

| Properti | Nilai |
|----------|-------|
| Nama token | SUBVOX |
| Jaringan | Solana (devnet untuk pengujian, mainnet TBD) |
| Alamat token | Segera hadir |
| Desimal | 6 |

### Pembagian pendapatan per terjemahan

| Bagian | Siapa yang mendapatkannya | Untuk apa |
|--------|--------------------------|-----------|
| **60%** | Penyedia | Dibagi secara proporsional di seluruh layanan yang digunakan: Groq (transkripsi), DeepSeek (terjemahan), OpenRouter (visi), cookie (akses platform) |
| **10%** | Platform | Hosting, pengembangan, operasional |
| **20%** | Kumpulan pemegang | Dibagikan setiap hari kepada semua pemegang token SUBVOX |
| **5%** | Dibakar | Token dihapus secara permanen dari peredaran |
| **5%** | Biaya operasional | Biaya infrastruktur, relay API, pemeliharaan |

Distribusi terjadi **secara otomatis** setiap 24 jam melalui cron job. Tanpa campur tangan manual.

**Contoh:** Pekerjaan yang menggunakan Groq + DeepSeek membagi 60% secara merata (30% + 30%). Pekerjaan yang menggunakan keempat layanan membagi secara proporsional (25% + 25% + 5% + 5%).

### Cara mendapatkan SUBVOX

| Metode | Deskripsi |
|--------|-----------|
| **Kontribusi kode** | Kirimkan PR ke [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) |
| **Bagikan kunci Groq** | Tambahkan kunci Anda ke kumpulan, dapatkan bagian dari kumpulan penyedia 60% |
| **Bagikan cookie** | Buka platform terbatas dan dapatkan imbalan |
| **Laporkan bug** | Laporan bug yang valid mendapatkan token |
| **Pekerjaan komunitas** | Terjemahan, dokumentasi, desain, moderasi |

> **Penting:** SUBVOX adalah token utilitas komunitas. Tidak memiliki nilai uang yang dijamin. Jangan pernah membayar uang untuk SUBVOX.

## Arsitektur

Untuk pengembang yang ingin mendalami kode:

```
Frontend (Next.js)  -->  Pipeline API (FastAPI)  -->  Celery Worker
                                                         |
                                                         |--> Unduh (yt-dlp)
                                                         |--> Transkripsi (Groq Whisper)
                                                         |--> Terjemahkan (DeepSeek + LLaMA)
                                                         |--> Bakar teks (FFmpeg)
                                                         |--> Kirim
```
