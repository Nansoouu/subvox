# Subvox — Italiano README

Read the main English README for the full version. Here is the key information in Italian.

## Cos'e Subvox?

Subvox e un **motore di sottotitoli open-source** per la comunita. Incolla un URL video, scegli una lingua e ottieni un video sottotitolato in pochi minuti.

**Oltre 1800 piattaforme supportate** — X/Twitter, TikTok, YouTube, Instagram, Vimeo, Dailymotion e migliaia di altre via yt-dlp.

## Come funziona

1. **Incolla un URL** — da una qualsiasi delle 1800+ piattaforme
2. **Scegli una lingua** — 21 lingue disponibili
3. **Aspetta 1-10 minuti** — la pipeline fa tutto
4. **Ottieni il tuo video** — con sottotitoli bruciati + SRT/VTT scaricabili

## Come guadagnare

Subvox e progettato in modo che **tutti possano guadagnare** contribuendo all'ecosistema, senza saper programmare.

**1. Condividi la tua chiave Groq** — Presta i tuoi crediti Groq al pool comunitario e guadagna SUBTEST proporzionalmente all'utilizzo.

**2. Condividi i tuoi cookie** — Alcuni siti (YouTube...) richiedono cookie. Condividili in modo sicuro e ottieni ricompense.

**3. Detieni SUBTEST** — I possessori di token ricevono una parte delle entrate della piattaforma.
