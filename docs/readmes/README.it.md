# Subvox -- Italiano README

Read the main English README for the full version. Here is the key information in Italian.

## Cos'e Subvox?

Subvox e un **motore di sottotitoli open-source** per la comunita. Incolla un URL video, scegli una lingua e ottieni un video sottotitolato in pochi minuti.

**Oltre 1800 piattaforme supportate** via yt-dlp: X/Twitter, TikTok, YouTube, Instagram, Vimeo, Dailymotion e migliaia di altre.

## Come funziona la pipeline

Ogni traduzione attraversa 5 fasi. La comunita interviene in ciascuna:

### 1. Download
Download via yt-dlp. I membri possono condividere cookie crittografati per piattaforme limitate.

### 2. Transcribe
Trascrizione via Groq Whisper. Il **pool comunitario Groq** fornisce crediti gratuiti.

### 3. Translate
Traduzione dei sottotitoli via DeepSeek.

### 4. Burn
Incorpora i sottotitoli nel video via FFmpeg.

### 5. Deliver
Il tuo video sottotitolato e pronto, con file SRT/VTT scaricabili.

## Economia comunitaria

| Quota | Chi riceve | Scopo |
|-------|-----------|-------|

## Come guadagnare

**1. Condividi la tua chiave Groq** -- Guadagna il 75% di ogni traduzione che usa la tua chiave.

**2. Condividi i tuoi cookie** -- Sblocca piattaforme limitate e guadagna SUBVOX.

**3. Detieni SUBVOX** -- Gli holder ricevono il 10% delle commissioni, distribuito giornalmente.
