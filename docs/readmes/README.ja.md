## Subvoxの理由

動画翻訳は**何十億人もの人々**に影響を与えています。韓国語のチュートリアル、ドイツ語のカンファレンス、アラビア語の講義、スペイン語のブログ。言語が障壁になるべきではありません。

**問題点：** 既存のツール（Veed、Kapwing、Descript、Opus Clip）は数分に制限し、エクスポートを制限し、高額なサブスクリプションに縛り付けます。アクセス可能であるべきものに法外な料金を請求します。

**解決策：** 全員が利益を得る分散型コミュニティ。

| 対象 | 得られるもの |
|-----|---------------|
| **ヘビーユーザー** | 競合他社の価格の一部で無制限の動画を翻訳。時間制限なし。21言語。 |
| **Groqキー保有者** | 使用していないキーを共有し、60%のプロバイダープールの分け前を獲得。不労所得。 |
| **トークン保有者** | 50万以上のSUBVOXをステークし、20%の保有者プールから毎日報酬を獲得。 |
| **バグ修正者＆貢献者** | バグを報告し、コードを書き、言語を翻訳。実際の金銭的価値を持つSUBVOXを獲得。 |
| **全員** | 透明性があり、監査可能で、オンチェーン。全員が報酬を受け取り、システムが機能することを保証します。 |

サブスクリプションなし。隠れた手数料なし。企業の思惑なし。コミュニティによって、コミュニティのために構築されています。

## パイプラインの仕組み

すべての翻訳は5つのステップを経ます。コミュニティが各ステップをどのように**強化**するかをご紹介します。

> 💡 **内部の仕組みを見たいですか？** パイプラインは完全にオープンソースで、[github.com/Nansoouu/subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) にあります。自分で実行し、すべてのコード行を検査し、貢献することができます。

### 1. ダウンロード
動画のURLはyt-dlpを介してダウンロードされます。コミュニティメンバーは暗号化されたクッキーを共有して、制限付きプラットフォーム（YouTubeなど）のロックを解除できます。

**あなたの収入：** クッキーを共有し、使用されるたびに報酬を得られます。

### 2. 文字起こし
音声はGroq Whisperを介してテキストに文字起こしされます。**コミュニティGroqプール**が無料の文字起こしクレジットを提供します。

**あなたの収入：** Groq APIキーをプールと共有します。プロバイダーシェア（翻訳コストの60%、使用されたすべてのサービスに分割）はキープロバイダーに支払われます。

### 3. 翻訳
字幕はDeepSeekを介してターゲット言語に翻訳されます（ローカルLLaMAへのフォールバックを計画中）。

### 4. 焼き付け
字幕はFFmpegを介して動画に焼き付けられます。長い動画向けのGPUサポートを計画中。

### 5. 配信
字幕付き動画がダウンロード可能なSRT/VTTファイルとともに準備完了です。

## 収入を得る

Subvoxは、**誰でも**エコシステムに貢献することで収入を得られるように設計されています。コーディングは不要です。

### 1. Groqキーを共有する

Groq APIキーをコミュニティプールに追加し、キーが文字起こしに使用されるたびに**60%のプロバイダープール**の分け前を獲得します。

| キーの容量 | 収入 |
|--------------------|---------------|
| 個人50% / 共有50% | キーの半分は自分用、半分はプール用（両方で収入） |
| 1日30分制限 | 最大30分/日は自分用に確保、残りは共有 |
| 無制限 | 完全な個人アクセス + 共有部分のプール報酬 |

**方法：** Webアプリ（近日公開予定）でウォレットを接続し、Billing > Groq Poolに移動し、キーを貼り付け、個人比率を設定します。

### 2. クッキーを共有する

一部のプラットフォーム（YouTubeなど）では、動画をダウンロードするために認証クッキーが必要です。暗号化されたクッキーを共有し、使用されるたびにSUBVOXを獲得します。

**方法：** クッキーはFernet（AES）で暗号化され、安全に保存されます。いつでもアクセスを無効にできます。

### 3. SUBVOXトークンを保有する

SUBVOXトークンを保有して、毎日分配される**20%の保有者プール**の分け前を獲得します。

| ティア | 最低残高 | 報酬の重み | 特典 |
|------|----------------|---------------|----------|
| **Decouverte** | 0 SUBVOX | 報酬なし | 2分未満の動画は無料、透かしあり、1日3回 |
| **Passion** | 500,000 SUBVOX | 1倍の重み | 無制限の時間、透かしなし、SRTエクスポート |
| **Builder** | 1,000,000 SUBVOX + プロバイダーキー | 2倍の重み | 2倍の報酬、プロバイダーキー収入にボーナス |

**仕組み：** 24時間ごとに、保有者プールは比例配分されます。加重されたSUBVOXの10%を保有している場合、その日のプールの10%を獲得します。

## SUBVOXトークン＆ティア

### 料金

翻訳コストは動画の長さに基づいて計算されます。価格は言語ごとにSUBVOXで表示されます。

| 動画の長さ | 価格（SUBVOX） |
|---------------|----------------|
| < 2分 | **無料**（Decouverteティア） |
| 2～5分 | 500 |
| 5～15分 | 1,500 |
| 15～30分 | 3,000 |
| 30～60分 | 6,000 |
| 60～90分 | 10,000 |

**多言語割引：** 追加言語ごとに20%オフ。

### トークン

| プロパティ | 値 |
|----------|-------|
| トークン名 | SUBVOX |
| ネットワーク | Solana（テスト用devnet、メインネットは未定） |
| トークンアドレス | 近日公開予定 |
| 小数桁数 | 6 |

### 翻訳ごとの収益配分

| シェア | 受取人 | 用途 |
|-------|------------|----------|
| **60%** | プロバイダー | 使用されたサービスに比例配分：Groq（文字起こし）、DeepSeek（翻訳）、OpenRouter（ビジョン）、クッキー（プラットフォームアクセス） |
| **10%** | プラットフォーム | ホスティング、開発、運用 |
| **20%** | 保有者プール | すべてのSUBVOXトークン保有者に毎日分配 |
| **5%** | バーン | トークンは永久に流通から除外 |
| **5%** | 運用手数料 | インフラコスト、APIリレー、メンテナンス |

この分配は、cronジョブを介して**24時間ごとに自動的に**行われます。手動操作は不要です。

**例：** Groq + DeepSeekを使用するジョブは、60%を均等に分割します（30% + 30%）。4つのサービスすべてを使用するジョブは、比例配分します（25% + 25% + 5% + 5%）。

### SUBVOXの入手方法

| 方法 | 説明 |
|--------|-------------|
| **コードの貢献** | [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) にPRを送信 |
| **Groqキーの共有** | キーをプールに追加し、60%のプロバイダープールの分け前を獲得 |
| **クッキーの共有** | 制限付きプラットフォームのロックを解除し、報酬を獲得 |
| **バグの報告** | 有効なバグ報告はトークンを獲得 |
| **コミュニティ活動** | 翻訳、ドキュメント、デザイン、モデレーション |

> **重要：** SUBVOXはコミュニティユーティリティトークンです。保証された金銭的価値はありません。SUBVOXにお金を支払わないでください。

## アーキテクチャ

コードを詳しく調べたい開発者向け：

```
フロントエンド (Next.js)  -->  パイプラインAPI (FastAPI)  -->  Celery Worker

                                                         |--> ダウンロード (yt-dlp)
                                                         |--> 文字起こし (Groq Whisper)
                                                         |--> 翻訳 (DeepSeek + LLaMA)

                                                         |--> 配信
```
