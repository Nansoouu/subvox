## Subvox를 선택해야 하는 이유는?

비디오 번역은 **수십억 명의 사람들**에게 영향을 미칩니다. 한국어 튜토리얼, 독일어 컨퍼런스, 아랍어 강의, 스페인어 브이로그. 언어는 절대 장벽이 되어서는 안 됩니다.

**문제점:** 기존 도구(Veed, Kapwing, Descript, Opus Clip)는 몇 분으로 제한하고, 내보내기를 제한하며, 비싼 구독에 가두어 놓습니다. 접근 가능해야 할 서비스에 막대한 비용을 청구합니다.

**해결책:** 모두가 이익을 얻는 분산형 커뮤니티입니다.

| 대상 | 혜택 |
|-----|---------------|
| **헤비 유저** | 경쟁사 가격의 극히 일부로 무제한 비디오를 번역하세요. 시간 제한 없음. 21개 언어 지원. |
| **Groq 키 보유자** | 사용하지 않는 키를 공유하고 60% 제공자 풀의 지분을 획득하세요. 수동적 소득. |
| **토큰 보유자** | 50만 개 이상의 SUBVOX를 스테이킹하고 20% 보유자 풀에서 매일 보상을 받으세요. |
| **버그 수정자 및 기여자** | 버그를 신고하고, 코드를 작성하고, 언어를 번역하세요. 실제 금전적 가치가 있는 SUBVOX를 획득하세요. |
| **모든 사람** | 투명하고, 감사 가능하며, 온체인 기반입니다. 우리는 모든 사람이 보상을 받고 시스템이 작동함을 인증합니다. |

구독 없음. 숨은 수수료 없음. 기업 의제 없음. 커뮤니티에 의해, 커뮤니티를 위해 구축되었습니다.

## 파이프라인 작동 방식

모든 번역은 5단계를 거칩니다. **커뮤니티**가 각 단계를 어떻게 지원하는지 소개합니다.

> 💡 **내부 작동 방식을 보고 싶으신가요?** 파이프라인은 [github.com/Nansoouu/subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline)에서 완전히 오픈소스로 제공됩니다. 직접 실행하고, 모든 코드 라인을 검사하고, 기여할 수 있습니다.

### 1. 다운로드
비디오 URL이 yt-dlp를 통해 다운로드됩니다. 커뮤니티 회원은 암호화된 쿠키를 공유하여 제한된 플랫폼(YouTube 등)의 잠금을 해제할 수 있습니다.

**수익 창출:** 쿠키를 공유하고 사용될 때마다 보상을 받으세요.

### 2. 전사
음성이 Groq Whisper를 통해 텍스트로 전사됩니다. **커뮤니티 Groq 풀**이 무료 전사 크레딧을 제공합니다.

**수익 창출:** Groq API 키를 풀에 공유하세요. 제공자 지분(번역 비용의 60%, 사용된 모든 서비스에 분할)이 키 제공자에게 돌아갑니다.

### 3. 번역
자막이 DeepSeek를 통해 대상 언어로 번역됩니다(로컬 LLaMA 폴백 계획 중).

### 4. 굽기
자막이 FFmpeg를 통해 비디오에 굽혀집니다. 긴 비디오를 위한 GPU 지원이 계획되어 있습니다.

### 5. 전달
자막이 포함된 비디오가 다운로드 가능한 SRT/VTT 파일과 함께 준비됩니다.

## 수익 창출

Subvox는 **누구나** 생태계에 기여하여 수익을 창출할 수 있도록 설계되었습니다. 코딩이 필요하지 않습니다.

### 1. Groq 키 공유

Groq API 키를 커뮤니티 풀에 추가하고 키가 전사에 사용될 때마다 **60% 제공자 풀**의 지분을 획득하세요.

| 키의 용량 | 획득 금액 |
|--------------------|---------------|
| 개인 50% / 공유 50% | 키의 절반은 본인용, 절반은 풀용 (둘 다 수익 창출) |
| 일일 30분 제한 | 최대 30분/일 본인 예약, 나머지는 공유 |
| 무제한 | 전체 개인 액세스 + 공유 부분에 대한 전체 풀 보상 |

**방법:** 웹 앱(곧 출시 예정)에서 지갑을 연결하고, 청구 > Groq 풀로 이동하여 키를 붙여넣고 개인 비율을 설정하세요.

### 2. 쿠키 공유

일부 플랫폼(YouTube 등)은 비디오를 다운로드하기 위해 인증 쿠키가 필요합니다. 암호화된 쿠키를 공유하고 사용될 때마다 SUBVOX를 획득하세요.

**방법:** 쿠키는 Fernet(AES)으로 암호화되어 안전하게 저장됩니다. 언제든지 액세스를 취소할 수 있습니다.

### 3. SUBVOX 토큰 보유

SUBVOX 토큰을 보유하여 매일 분배되는 **20% 보유자 풀**의 지분을 획득하세요.

| 등급 | 최소 잔액 | 보상 가중치 | 혜택 |
|------|----------------|---------------|----------|
| **Decouverte** | 0 SUBVOX | 보상 없음 | 2분 미만 비디오 무료, 워터마크, 하루 3회 |
| **Passion** | 500,000 SUBVOX | 1배 가중치 | 무제한 시간, 워터마크 없음, SRT 내보내기 |
| **Builder** | 1,000,000 SUBVOX + 제공자 키 | 2배 가중치 | 2배 보상, 제공자 키 수익에 보너스 |

**작동 방식:** 24시간마다 보유자 풀이 비례적으로 분할됩니다. 가중치가 적용된 SUBVOX의 10%를 보유하면 해당 일일 풀의 10%를 받습니다.

## SUBVOX 토큰 및 등급

### 가격

번역 비용은 비디오 길이에 따라 계산됩니다. 가격은 언어당 SUBVOX로 표시됩니다.

| 비디오 길이 | 가격 (SUBVOX) |
|---------------|----------------|
| < 2분 | **무료** (Decouverte 등급) |
| 2-5분 | 500 |
| 5-15분 | 1,500 |
| 15-30분 | 3,000 |
| 30-60분 | 6,000 |
| 60-90분 | 10,000 |

**다국어 할인:** 추가 언어당 20% 할인.

### 토큰

| 속성 | 값 |
|----------|-------|
| 토큰 이름 | SUBVOX |
| 네트워크 | Solana (테스트용 devnet, 메인넷 TBD) |
| 토큰 주소 | 곧 출시 예정 |
| 소수점 자리 | 6 |

### 번역당 수익 분배

| 지분 | 수취인 | 용도 |
|-------|------------|----------|
| **60%** | 제공자 | 사용된 서비스에 비례 분할: Groq(전사), DeepSeek(번역), OpenRouter(비전), 쿠키(플랫폼 액세스) |
| **10%** | 플랫폼 | 호스팅, 개발, 운영 |
| **20%** | 보유자 풀 | 모든 SUBVOX 토큰 보유자에게 매일 분배 |
| **5%** | 소각 | 토큰이 유통에서 영구적으로 제거됨 |
| **5%** | 운영 수수료 | 인프라 비용, API 릴레이, 유지보수 |

분배는 크론 작업을 통해 **24시간마다 자동으로** 이루어집니다. 수동 개입이 필요하지 않습니다.

**예시:** Groq + DeepSeek를 사용하는 작업은 60%를 균등하게 분할합니다(30% + 30%). 네 가지 서비스를 모두 사용하는 작업은 비례적으로 분할합니다(25% + 25% + 5% + 5%).

### SUBVOX 획득 방법

| 방법 | 설명 |
|--------|-------------|
| **코드 기여** | [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline)에 PR 제출 |
| **Groq 키 공유** | 키를 풀에 추가하고 60% 제공자 풀의 지분 획득 |
| **쿠키 공유** | 제한된 플랫폼 잠금 해제 및 보상 획득 |
| **버그 신고** | 유효한 버그 신고는 토큰 획득 |
| **커뮤니티 작업** | 번역, 문서, 디자인, 중재 |

> **중요:** SUBVOX는 커뮤니티 유틸리티 토큰입니다. 보장된 금전적 가치는 없습니다. SUBVOX에 대해 절대 돈을 지불하지 마십시오.

## 아키텍처

코드를 살펴보고 싶은 개발자를 위한 설명입니다:

```
프론트엔드 (Next.js)  -->  파이프라인 API (FastAPI)  -->  Celery Worker
                                                         |
                                                         |--> 다운로드 (yt-dlp)
                                                         |--> 전사 (Groq Whisper)
                                                         |--> 번역 (DeepSeek + LLaMA)
                                                         |--> 자막 굽기 (FFmpeg)
                                                         |--> 전달
```
