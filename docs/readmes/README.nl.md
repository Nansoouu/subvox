## Waarom Subvox?

Videovertaling raakt **miljarden mensen**. Een Koreaanse tutorial, een Duitse conferentie, een Arabische lezing, een Spaanse vlog. Taal mag nooit een barrière zijn.

**Het probleem:** Bestaande tools (Veed, Kapwing, Descript, Opus Clip) beperken je tot een paar minuten, beperken je exports en sluiten je op in dure abonnementen. Ze rekenen een fortuin voor wat toegankelijk zou moeten zijn.

**De oplossing:** Een gedecentraliseerde gemeenschap waar iedereen wint.

| Wie | Wat ze krijgen |
|-----|---------------|
| **Zware gebruikers** | Vertaal onbeperkt video's voor een fractie van de prijs van concurrenten. Geen tijdslimieten. 21 talen. |
| **Groq-sleutelhouders** | Deel sleutels die je niet gebruikt en verdien een deel van de 60% providerpool. Passief inkomen. |
| **Tokenhouders** | Stak 500K+ SUBVOX en verdien dagelijkse beloningen uit de 20% houderspool. |
| **Bugfixers & bijdragers** | Meld bugs, schrijf code, vertaal talen. Verdien SUBVOX met echte geldwaarde. |
| **Iedereen** | Transparant, controleerbaar, on-chain. Wij garanderen dat iedereen betaald krijgt en het systeem werkt. |

Geen abonnement. Geen verborgen kosten. Geen bedrijfsagenda. Gebouwd door de gemeenschap, voor de gemeenschap.

## Hoe de pipeline werkt

Elke vertaling doorloopt 5 stappen. Hier is hoe de **gemeenschap** elke stap mogelijk maakt.

> 💡 **Wil je zien hoe het onder de motorkap werkt?** De pipeline is volledig open-source op [github.com/Nansoouu/subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline). Je kunt het zelf draaien, elke regel code inspecteren en bijdragen.

### 1. Downloaden
Je video-URL wordt gedownload via yt-dlp. Gemeenschapsleden kunnen versleutelde cookies delen om beperkte platforms (YouTube, enz.) te ontgrendelen.

**Jij verdient:** Deel je cookies en word elke keer dat ze worden gebruikt beloond.

### 2. Transcriberen
Spraak wordt omgezet naar tekst via Groq Whisper. De **gemeenschaps-Groq-pool** biedt gratis transcriptiekredieten.

**Jij verdient:** Deel je Groq API-sleutel met de pool. Het provideraandeel (60% van de vertaalkosten, verdeeld over alle gebruikte diensten) gaat naar de sleutelproviders.

### 3. Vertalen
Ondertitels worden vertaald naar de doeltaal via DeepSeek (met lokale LLaMA-fallback gepland).

### 4. Branden
Ondertitels worden in de video gebrand via FFmpeg. GPU-ondersteuning voor lange video's is gepland.

### 5. Leveren
Je video met ondertiteling is klaar met downloadbare SRT/VTT-bestanden.

## Verdienen

Subvox is ontworpen zodat **iedereen kan verdienen** door bij te dragen aan het ecosysteem. Geen codeervaardigheden vereist.

### 1. Deel je Groq-sleutel

Voeg je Groq API-sleutel toe aan de gemeenschapspool en verdien een deel van de **60% providerpool** elke keer dat je sleutel wordt gebruikt voor transcriptie.

| Capaciteit van je sleutel | Wat je verdient |
|--------------------|---------------|
| 50% persoonlijk / 50% gedeeld | De helft van je sleutel voor jezelf, de helft voor de pool (je verdient op beide) |
| 30 min dagelijkse limiet | Tot 30 min/dag voor jou gereserveerd, de rest gedeeld |
| Onbeperkt | Volledige persoonlijke toegang + volledige poolbeloningen op het gedeelde deel |

**Hoe:** Verbind je wallet in de webapp (binnenkort beschikbaar), ga naar Facturatie > Groq Pool, plak je sleutel, stel je persoonlijke verhouding in.

### 2. Deel je cookies

Sommige platforms (YouTube, enz.) hebben authenticatiecookies nodig om video's te downloaden. Deel je versleutelde cookies en verdien SUBVOX elke keer dat ze worden gebruikt.

**Hoe:** Je cookies worden versleuteld met Fernet (AES) en veilig opgeslagen. Je kunt de toegang op elk moment intrekken.

### 3. Houd SUBVOX-tokens vast

Houd SUBVOX-tokens vast om een deel van de **20% houderspool** te verdienen, dagelijks uitbetaald.

| Niveau | Minimumsaldo | Beloningsgewicht | Voordelen |
|------|----------------|---------------|----------|
| **Decouverte** | 0 SUBVOX | Geen beloningen | Gratis voor video's < 2 min, watermerk, 3/dag |
| **Passion** | 500.000 SUBVOX | 1x gewicht | Onbeperkte duur, geen watermerk, SRT-export |
| **Builder** | 1.000.000 SUBVOX + providersleutel | 2x gewicht | 2x beloningen, bonus op je providersleutelinkomsten |

**Hoe het werkt:** Elke 24 uur wordt de houderspool proportioneel verdeeld. Als je 10% van alle gewogen SUBVOX bezit, krijg je 10% van die dagelijkse pool.

## SUBVOX-token & niveaus

### Prijzen

Vertaalkosten worden berekend op basis van videoduur. Prijzen zijn in SUBVOX, per taal.

| Videoduur | Prijs (SUBVOX) |
|---------------|----------------|
| < 2 min | **Gratis** (Decouverte-niveau) |
| 2-5 min | 500 |
| 5-15 min | 1.500 |
| 15-30 min | 3.000 |
| 30-60 min | 6.000 |
| 60-90 min | 10.000 |

**Meertalige korting:** 20% korting op elke extra taal.

### Token

| Eigenschap | Waarde |
|----------|-------|
| Tokennaam | SUBVOX |
| Netwerk | Solana (devnet voor testen, mainnet TBD) |
| Tokenadres | Binnenkort beschikbaar |
| Decimalen | 6 |

### Inkomstenverdeling per vertaling

| Aandeel | Wie krijgt het | Waarvoor |
|-------|------------|----------|
| **60%** | Providers | Proportioneel verdeeld over gebruikte diensten: Groq (transcriptie), DeepSeek (vertaling), OpenRouter (visie), cookies (platformtoegang) |
| **10%** | Platform | Hosting, ontwikkeling, operaties |
| **20%** | Houderspool | Dagelijks verdeeld onder alle SUBVOX-tokenhouders |
| **5%** | Verbrand | Tokens permanent uit circulatie verwijderd |
| **5%** | Operationele kosten | Infrastructuurkosten, API-relay, onderhoud |

De verdeling gebeurt **automatisch** elke 24 uur via een cron-job. Geen handmatige tussenkomst.

**Voorbeeld:** Een taak die Groq + DeepSeek gebruikt, verdeelt de 60% gelijk (30% + 30%). Een taak die alle vier de diensten gebruikt, verdeelt proportioneel (25% + 25% + 5% + 5%).

### Hoe krijg je SUBVOX

| Methode | Beschrijving |
|--------|-------------|
| **Code bijdragen** | Dien een PR in bij [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) |
| **Een Groq-sleutel delen** | Voeg je sleutel toe aan de pool, verdien een deel van de 60% providerpool |
| **Cookies delen** | Ontgrendel beperkte platforms en verdien beloningen |
| **Bugs melden** | Geldige bugmeldingen leveren tokens op |
| **Gemeenschapswerk** | Vertalingen, documentatie, ontwerp, moderatie |

> **Belangrijk:** SUBVOX is een gemeenschapsnutstoken. Het heeft geen gegarandeerde geldwaarde. Betaal nooit geld voor SUBVOX.

## Architectuur

Voor ontwikkelaars die in de code willen duiken:

```
Frontend (Next.js)  -->  Pipeline API (FastAPI)  -->  Celery Worker
                                                         |
                                                         |--> Downloaden (yt-dlp)
                                                         |--> Transcriberen (Groq Whisper)
                                                         |--> Vertalen (DeepSeek + LLaMA)
                                                         |--> Ondertitels branden (FFmpeg)
                                                         |--> Leveren
```

| Repository | Zichtbaarheid | Doel |
|-----------|-----------|---------|
| [subvox](https://github.com/Nansoouu/subvox) | Openbaar | **Deze repo** -- gemeenschapshub, documentatie, pools |
| [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) | Openbaar | Pipeline-engine (open-source) |
