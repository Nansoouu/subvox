# Subvox -- Portuguese README

Read the main English README for the full version. Here is the key information in Portuguese.

## O que e Subvox?

Subvox e um **motor de legendas open-source** para a comunidade. Cole uma URL de video, escolha um idioma, e obtenha um video legendado em minutos.

**Mais de 1800 plataformas suportadas** via yt-dlp: X/Twitter, TikTok, YouTube, Instagram, Vimeo, Dailymotion, e milhares mais.

## Como o pipeline funciona

Cada traducao passa por 5 etapas. A comunidade intervem em cada uma:

### 1. Download
Download via yt-dlp. Membros podem compartilhar cookies criptografados para plataformas restritas.

### 2. Transcribe
Transcricao via Groq Whisper. O **pool comunitario Groq** fornece creditos gratuitos.

### 3. Translate
Traducao de legendas via DeepSeek.

### 4. Burn
Legendas incorporadas no video via FFmpeg.

### 5. Deliver
Seu video legendado esta pronto, com arquivos SRT/VTT para download.

**1. Compartilhe sua chave Groq** -- Ganhe 75% de cada traducao que usar sua chave.
