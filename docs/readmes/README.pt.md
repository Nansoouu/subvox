## Por que Subvox?

A tradução de vídeos afeta **bilhões de pessoas**. Um tutorial coreano, uma conferência alemã, uma palestra em árabe, um vlog em espanhol. O idioma nunca deve ser uma barreira.

**O problema:** Ferramentas existentes (Veed, Kapwing, Descript, Opus Clip) limitam você a alguns minutos, restringem suas exportações e prendem você em assinaturas caras. Elas cobram uma fortuna pelo que deveria ser acessível.

**A solução:** Uma comunidade descentralizada onde todos ganham.

| Quem | O que recebem |
|------|---------------|
| **Usuários pesados** | Traduza vídeos ilimitados por uma fração dos preços dos concorrentes. Sem limites de tempo. 21 idiomas. |
| **Detentores de chave Groq** | Compartilhe chaves que você não está usando e ganhe uma parte do pool de 60% dos provedores. Renda passiva. |
| **Detentores de tokens** | Faça stake de 500K+ SUBVOX e ganhe recompensas diárias do pool de 20% dos detentores. |
| **Corretores de bugs e contribuidores** | Reporte bugs, escreva código, traduza idiomas. Ganhe SUBVOX que tem valor monetário real. |
| **Todos** | Transparente, auditável, on-chain. Certificamos que todos recebem pagamento e o sistema funciona. |

Sem assinatura. Sem taxas ocultas. Sem agenda corporativa. Construído pela comunidade, para a comunidade.

## Como o pipeline funciona

Cada tradução passa por 5 etapas. Veja como a **comunidade** impulsiona cada uma delas.

> 💡 **Quer ver como funciona por baixo dos panos?** O pipeline é totalmente open-source em [github.com/Nansoouu/subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline). Você pode executá-lo você mesmo, inspecionar cada linha de código e contribuir.

### 1. Download
Sua URL de vídeo é baixada via yt-dlp. Membros da comunidade podem compartilhar cookies criptografados para desbloquear plataformas restritas (YouTube, etc.).

**Você ganha:** Compartilhe seus cookies e seja recompensado cada vez que forem usados.

### 2. Transcrição
A fala é transcrita para texto via Groq Whisper. O **pool comunitário Groq** fornece créditos de transcrição gratuitos.

**Você ganha:** Compartilhe sua chave de API Groq com o pool. A parte do provedor (60% do custo da tradução, dividida entre todos os serviços usados) vai para os provedores de chave.

### 3. Tradução
As legendas são traduzidas para o idioma alvo via DeepSeek (com fallback local para LLaMA planejado).

### 4. Queima
As legendas são queimadas no vídeo via FFmpeg. Suporte a GPU para vídeos longos está planejado.

### 5. Entrega
Seu vídeo legendado está pronto com arquivos SRT/VTT para download.

## Ganhe

O Subvox é projetado para que **qualquer um possa ganhar** contribuindo para o ecossistema. Não é necessário saber programar.

### 1. Compartilhe sua chave Groq

Adicione sua chave de API Groq ao pool comunitário e ganhe uma parte do **pool de 60% dos provedores** toda vez que sua chave for usada para transcrição.

| Capacidade da sua chave | O que você ganha |
|------------------------|------------------|
| 50% pessoal / 50% compartilhado | Metade da sua chave para você, metade para o pool (você ganha em ambas) |
| Limite diário de 30 min | Até 30 min/dia reservados para você, o resto compartilhado |
| Ilimitado | Acesso pessoal completo + recompensas completas do pool na parte compartilhada |

**Como:** Conecte sua carteira no aplicativo web (em breve), vá em Faturamento > Pool Groq, cole sua chave, defina sua proporção pessoal.

### 2. Compartilhe seus cookies

Algumas plataformas (YouTube, etc.) exigem cookies de autenticação para baixar vídeos. Compartilhe seus cookies criptografados e ganhe SUBVOX cada vez que forem usados.

**Como:** Seus cookies são criptografados com Fernet (AES) e armazenados com segurança. Você pode revogar o acesso a qualquer momento.

### 3. Tenha tokens SUBVOX

Tenha tokens SUBVOX para ganhar uma parte do **pool de 20% dos detentores**, distribuído diariamente.

| Nível | Saldo mínimo | Peso da recompensa | Benefícios |
|-------|--------------|--------------------|------------|
| **Decouverte** | 0 SUBVOX | Sem recompensas | Grátis para vídeos < 2 min, marca d'água, 3/dia |
| **Passion** | 500.000 SUBVOX | Peso 1x | Duração ilimitada, sem marca d'água, exportação SRT |
| **Builder** | 1.000.000 SUBVOX + chave de provedor | Peso 2x | Recompensas 2x, bônus na receita da sua chave de provedor |

**Como funciona:** A cada 24 horas, o pool de detentores é dividido proporcionalmente. Se você detém 10% de todos os SUBVOX ponderados, você recebe 10% do pool daquele dia.

## Token SUBVOX e níveis

### Preços

Os custos de tradução são calculados pela duração do vídeo. Os preços são em SUBVOX, por idioma.

| Duração do vídeo | Preço (SUBVOX) |
|------------------|----------------|
| < 2 min | **Grátis** (nível Decouverte) |
| 2-5 min | 500 |
| 5-15 min | 1.500 |
| 15-30 min | 3.000 |
| 30-60 min | 6.000 |
| 60-90 min | 10.000 |

**Desconto para vários idiomas:** 20% de desconto em cada idioma adicional.

### Token

| Propriedade | Valor |
|-------------|-------|
| Nome do token | SUBVOX |
| Rede | Solana (devnet para testes, mainnet a definir) |
| Endereço do token | Em breve |
| Decimais | 6 |

### Divisão de receita por tradução

| Parte | Quem recebe | Para quê |
|-------|-------------|----------|
| **60%** | Provedores | Dividido proporcionalmente entre os serviços usados: Groq (transcrição), DeepSeek (tradução), OpenRouter (visão), cookies (acesso à plataforma) |
| **10%** | Plataforma | Hospedagem, desenvolvimento, operações |
| **20%** | Pool de detentores | Distribuído diariamente a todos os detentores de tokens SUBVOX |
| **5%** | Queimado | Tokens permanentemente removidos de circulação |
| **5%** | Taxas operacionais | Custos de infraestrutura, relay de API, manutenção |

A distribuição acontece **automaticamente** a cada 24 horas via um cron job. Sem intervenção manual.

**Exemplo:** Um trabalho usando Groq + DeepSeek divide os 60% igualmente (30% + 30%). Um trabalho usando todos os quatro serviços divide proporcionalmente (25% + 25% + 5% + 5%).

## Arquitetura

Para desenvolvedores que desejam mergulhar no código:

```
Frontend (Next.js)  -->  Pipeline API (FastAPI)  -->  Celery Worker
                                                         |
                                                         |--> Download (yt-dlp)
                                                         |--> Transcrição (Groq Whisper)
                                                         |--> Tradução (DeepSeek + LLaMA)
                                                         |--> Queima de legendas (FFmpeg)
                                                         |--> Entrega
```

| Repositório | Visibilidade | Propósito |
|-------------|--------------|-----------|
| [subvox](https://github.com/Nansoouu/subvox) | Público | **Este repositório** -- hub comunitário, documentação, pools |
| [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) | Público | Motor do pipeline (open-source) |
