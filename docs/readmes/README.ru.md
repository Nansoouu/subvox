## Зачем Subvox?

Перевод видео затрагивает **миллиарды людей**. Корейский урок, немецкая конференция, арабская лекция, испанский влог. Язык никогда не должен быть барьером.

**Проблема:** Существующие инструменты (Veed, Kapwing, Descript, Opus Clip) ограничивают вас несколькими минутами, лимитируют экспорт и запирают в дорогие подписки. Они берут огромные деньги за то, что должно быть доступным.

**Решение:** Децентрализованное сообщество, где выигрывают все.

| Кто | Что получает |
|-----|---------------|
| **Активные пользователи** | Переводите неограниченное количество видео за долю от цен конкурентов. Без временных лимитов. 135+ язык. |
| **Владельцы ключей Groq** | Делитесь ключами, которые вы не используете, и зарабатывайте долю пула провайдеров (60%). Пассивный доход. |
| **Держатели токенов** | Стейкайте 500K+ SUBVOX и получайте ежедневные вознаграждения из пула держателей (20%). |
| **Исправители багов и контрибьюторы** | Сообщайте об ошибках, пишите код, переводите языки. Зарабатывайте SUBVOX, имеющий реальную денежную ценность. |
| **Все** | Прозрачно, проверяемо, на блокчейне. Мы гарантируем, что каждый получает оплату и система работает. |

Никакой подписки. Никаких скрытых платежей. Никакой корпоративной повестки. Создано сообществом, для сообщества.

## Как работает пайплайн

Каждый перевод проходит через 5 шагов. Вот как **сообщество** обеспечивает работу каждого из них.

> 💡 **Хотите увидеть, как это работает под капотом?** Пайплайн полностью открыт на [github.com/Nansoouu/subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline). Вы можете запустить его самостоятельно, проверить каждую строку кода и внести свой вклад.

### 1. Загрузка
URL вашего видео загружается через yt-dlp. Участники сообщества могут делиться зашифрованными куки для разблокировки ограниченных платформ (YouTube и т.д.).

**Вы зарабатываете:** Делитесь своими куки и получайте вознаграждение каждый раз, когда они используются.

### 2. Расшифровка
Речь преобразуется в текст через Groq Whisper. **Общественный пул Groq** предоставляет бесплатные кредиты на расшифровку.

**Вы зарабатываете:** Делитесь своим API-ключом Groq с пулом. Доля провайдера (60% стоимости перевода, распределяемая между всеми используемыми сервисами) достаётся владельцам ключей.

### 3. Перевод
Субтитры переводятся на целевой язык через DeepSeek (в будущем планируется локальный LLaMA как запасной вариант).

### 4. Встраивание
Субтитры встраиваются в видео через FFmpeg. Планируется поддержка GPU для длинных видео.

### 5. Доставка
Ваше видео с субтитрами готово, доступны файлы SRT/VTT для скачивания.

## Заработок

Subvox создан так, чтобы **каждый мог зарабатывать**, внося вклад в экосистему. Кодить не обязательно.

### 1. Поделитесь своим ключом Groq

Добавьте свой API-ключ Groq в общественный пул и зарабатывайте долю **пула провайдеров (60%)** каждый раз, когда ваш ключ используется для расшифровки.

| Ёмкость вашего ключа | Что вы зарабатываете |
|--------------------|---------------|
| 50% лично / 50% в общем доступе | Половина ключа для себя, половина для пула (вы зарабатываете на обоих) |
| Ежедневный лимит 30 мин | До 30 мин/день зарезервировано для вас, остальное в общем доступе |
| Безлимитный | Полный личный доступ + полные вознаграждения от пула за общую часть |

**Как:** Подключите свой кошелёк в веб-приложении (скоро), перейдите в Billing > Groq Pool, вставьте свой ключ, установите личное соотношение.

### 2. Поделитесь своими куки

Некоторые платформы (YouTube и т.д.) требуют куки аутентификации для скачивания видео. Делитесь своими зашифрованными куки и зарабатывайте SUBVOX каждый раз, когда они используются.

**Как:** Ваши куки шифруются с помощью Fernet (AES) и хранятся безопасно. Вы можете отозвать доступ в любое время.

### 3. Владейте токенами SUBVOX

Владейте токенами SUBVOX, чтобы получать долю **пула держателей (20%)**, распределяемого ежедневно.

| Уровень | Минимальный баланс | Вес вознаграждения | Преимущества |
|------|----------------|---------------|----------|
| **Decouverte** | 0 SUBVOX | Без вознаграждений | Бесплатно для видео < 2 мин, водяной знак, 3/день |
| **Passion** | 500,000 SUBVOX | Вес 1x | Безлимитная длительность, без водяного знака, экспорт SRT |
| **Builder** | 1,000,000 SUBVOX + ключ провайдера | Вес 2x | Вознаграждения x2, бонус к доходу от вашего ключа провайдера |

**Как это работает:** Каждые 24 часа пул держателей делится пропорционально. Если вы владеете 10% всех взвешенных SUBVOX, вы получаете 10% пула за этот день.

## Токен SUBVOX и уровни

### Ценообразование

Стоимость перевода рассчитывается исходя из длительности видео. Цены указаны в SUBVOX, за язык.

| Длительность видео | Цена (SUBVOX) |
|---------------|----------------|
| < 2 мин | **Бесплатно** (уровень Decouverte) |
| 2-5 мин | 500 |
| 5-15 мин | 1,500 |
| 15-30 мин | 3,000 |
| 30-60 мин | 6,000 |
| 60-90 мин | 10,000 |

**Скидка за несколько языков:** 20% за каждый дополнительный язык.

### Токен

| Свойство | Значение |
|----------|-------|
| Название токена | SUBVOX |
| Сеть | Solana (devnet для тестирования, mainnet уточняется) |
| Адрес токена | Скоро |
| Десятичные знаки | 6 |

### Распределение дохода за перевод

| Доля | Кто получает | За что |
|-------|------------|----------|
| **60%** | Провайдеры | Пропорционально распределяется между используемыми сервисами: Groq (расшифровка), DeepSeek (перевод), OpenRouter (зрение), куки (доступ к платформам) |
| **10%** | Платформа | Хостинг, разработка, эксплуатация |
| **20%** | Пул держателей | Распределяется ежедневно среди всех держателей токенов SUBVOX |
| **5%** | Сжигается | Токены навсегда изымаются из обращения |
| **5%** | Операционные расходы | Инфраструктурные затраты, ретрансляция API, обслуживание |

Распределение происходит **автоматически** каждые 24 часа через cron-задачу. Без ручного вмешательства.

**Пример:** Задача, использующая Groq + DeepSeek, делит 60% поровну (30% + 30%). Задача, использующая все четыре сервиса, делит пропорционально (25% + 25% + 5% + 5%).

### Как получить SUBVOX

| Метод | Описание |
|--------|-------------|
| **Внести код** | Отправьте PR в [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) |
| **Поделиться ключом Groq** | Добавьте свой ключ в пул, зарабатывайте долю пула провайдеров (60%) |
| **Поделиться куки** | Разблокируйте ограниченные платформы и получайте вознаграждения |
| **Сообщить об ошибке** | Подтверждённые отчёты об ошибках приносят токены |
| **Работа в сообществе** | Переводы, документация, дизайн, модерация |

> **Важно:** SUBVOX — это утилитарный токен сообщества. Он не имеет гарантированной денежной ценности. Никогда не платите деньги за SUBVOX.

## Архитектура

Для разработчиков, желающих погрузиться в код:

```
Фронтенд (Next.js)  -->  Pipeline API (FastAPI)  -->  Celery Worker
                                                         |
                                                         |--> Загрузка (yt-dlp)
                                                         |--> Расшифровка (Groq Whisper)
                                                         |--> Перевод (DeepSeek + LLaMA)
                                                         |--> Встраивание субтитров (FFmpeg)
                                                         |--> Доставка
```

| Репозиторий | Видимость | Назначение |
|-----------|-----------|---------|
| [subvox](https://github.com/Nansoouu/subvox) | Публичный | **Этот репозиторий** — центр сообщества, документация, пулы |
| [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) | Публичный | Движок пайплайна (открытый исходный код) |
