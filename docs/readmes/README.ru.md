Прочитайте основной README на английском для полной версии. Вот ключевая информация на русском.

Subvox это **движок перевода видео с открытым исходным кодом** для сообщества. Вставьте URL видео, выберите язык и получите видео с субтитрами за минуты.

## Почему Subvox?

Перевод видео касается **миллиардов людей**. Корейский урок, немецкая конференция, арабская лекция, испанский влог. Язык не должен быть барьером.

**Проблема:** Существующие инструменты (Veed, Kapwing, Descript, Opus Clip) ограничивают минуты, экспорт и запирают в дорогих подписках.

**Решение:** Децентрализованное сообщество, где выигрывают все.

| Кто | Что получает |
|-----|-------------|
| **Активные пользователи** | Неограниченный перевод по низкой цене. 21 язык. |
| **Владельцы ключей Groq** | Делитесь неиспользуемыми ключами и зарабатывайте 55% с каждого перевода. |
| **Держатели токенов** | Держите 500K+ SUBVOX и получайте ежедневные награды. |
| **Участники** | Сообщайте об ошибках, пишите код, переводите. Зарабатывайте SUBVOX. |
| **Все** | Прозрачная и проверяемая система. Мы гарантируем, что каждый получает оплату. |

## Как работает конвейер

### 1. Загрузка
Загрузка через yt-dlp. Делитесь зашифрованными куки и зарабатывайте SUBVOX при каждом использовании.

### 2. Транскрипция
Транскрипция через Groq Whisper. **Общественный пул Groq** предоставляет бесплатные кредиты. Делитесь ключом и зарабатывайте 55% стоимости.

### 3. Перевод
Перевод субтитров через DeepSeek.

### 4. Встраивание
Встраивание субтитров в видео через FFmpeg.

### 5. Доставка
Ваше видео с субтитрами готово с файлами SRT/VTT для скачивания.

## Экономика сообщества

Каждый перевод генерирует SUBVOX, которые автоматически распределяются:

| Доля | Кто получает |
|------|-------------|
| **55%** | Поставщик ключа Groq |
| **20%** | Платформа (хостинг, разработка, операции) |
| **20%** | Пул держателей (распределяется ежедневно) |
| **5%** | Сжигается (изымается из обращения) |

## Цены SUBVOX

| Длительность видео | Цена (SUBVOX) |
|-------------------|---------------|
| < 2 мин | **Бесплатно** (уровень Discovery) |
| 2-5 мин | 500 |
| 5-15 мин | 1,500 |
| 15-30 мин | 3,000 |
| 30-60 мин | 6,000 |
| 60-90 мин | 10,000 |

**Скидка на несколько языков:** 20% на каждый дополнительный язык.

## Как заработать

**1. Поделитесь ключом Groq** -- Зарабатывайте 55% с каждого перевода, использующего ваш ключ.

**2. Поделитесь куки** -- Разблокируйте ограниченные платформы и зарабатывайте SUBVOX.

**3. Держите SUBVOX** -- Держатели получают 10% комиссий, распределяемых ежедневно.

## Уровни

| Уровень | Мин. баланс | Вес наград | Преимущества |
|---------|------------|-----------|-------------|
