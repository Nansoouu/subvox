## Чому Subvox?

Переклад відео стосується **мільярдів людей**. Корейський туторіал, німецька конференція, арабська лекція, іспанський влог. Мова ніколи не повинна бути бар'єром.

**Проблема:** Існуючі інструменти (Veed, Kapwing, Descript, Opus Clip) обмежують вас кількома хвилинами, обмежують експорт і замикають у дорогих підписках. Вони беруть величезні гроші за те, що має бути доступним.

**Рішення:** Децентралізована спільнота, де всі виграють.

| Хто | Що отримує |
|-----|---------------|
| **Активні користувачі** | Перекладайте необмежену кількість відео за частку ціни конкурентів. Без обмежень за часом. 21 мова. |
| **Власники ключів Groq** | Діліться ключами, які ви не використовуєте, і заробляйте частку пулу постачальників (60%). Пасивний дохід. |
| **Власники токенів** | Стейкайте 500K+ SUBVOX і отримуйте щоденні винагороди з пулу власників (20%). |
| **Виправники помилок та контриб'ютори** | Повідомляйте про помилки, пишіть код, перекладайте мови. Заробляйте SUBVOX, які мають реальну грошову цінність. |
| **Всі** | Прозора, контрольована, ончейн-система. Ми гарантуємо, що всі отримують оплату, а система працює. |

Жодної підписки. Жодних прихованих комісій. Жодних корпоративних програм. Побудовано спільнотою, для спільноти.

## Як працює пайплайн

Кожен переклад проходить 5 кроків. Ось як **спільнота** забезпечує роботу кожного з них.

> 💡 **Хочете побачити, як це працює під капотом?** Пайплайн повністю відкритий за адресою [github.com/Nansoouu/subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline). Ви можете запустити його самостійно, перевірити кожен рядок коду та зробити свій внесок.

### 1. Завантаження
Ваше відео завантажується через yt-dlp. Члени спільноти можуть ділитися зашифрованими файлами cookie, щоб розблокувати обмежені платформи (YouTube тощо).

**Ви заробляєте:** Діліться своїми файлами cookie та отримуйте винагороду щоразу, коли вони використовуються.

### 2. Транскрипція
Мова транскрибується в текст за допомогою Groq Whisper. **Спільнотний пул Groq** надає безкоштовні кредити на транскрипцію.

**Ви заробляєте:** Діліться своїм ключем API Groq з пулом. Частка постачальника (60% вартості перекладу, розподілена між усіма використаними сервісами) надходить власникам ключів.

### 3. Переклад
Субтитри перекладаються цільовою мовою за допомогою DeepSeek (з планованим резервним варіантом локального LLaMA).

### 4. Вбудовування
Субтитри вбудовуються у відео за допомогою FFmpeg. Планується підтримка GPU для довгих відео.

### 5. Доставка
Ваше відео з субтитрами готове, доступні файли SRT/VTT для завантаження.

## Заробіток

Subvox створено так, щоб **кожен міг заробляти**, роблячи внесок в екосистему. Кодування не потрібне.

### 1. Поділіться своїм ключем Groq

Додайте свій ключ API Groq до спільнотного пулу та заробляйте частку **пулу постачальників (60%)** щоразу, коли ваш ключ використовується для транскрипції.

| Потужність вашого ключа | Що ви заробляєте |
|--------------------|---------------|
| 50% особисто / 50% спільно | Половина ключа для вас, половина для пулу (ви заробляєте на обох) |
| Щоденний ліміт 30 хв | До 30 хв/день зарезервовано для вас, решта спільна |
| Без обмежень | Повний особистий доступ + повні винагороди з пулу за спільну частину |

**Як:** Підключіть свій гаманець у веб-додатку (незабаром), перейдіть до Billing > Groq Pool, вставте свій ключ, встановіть своє особисте співвідношення.

### 2. Поділіться своїми файлами cookie

Деякі платформи (YouTube тощо) вимагають файли cookie автентифікації для завантаження відео. Поділіться своїми зашифрованими файлами cookie та заробляйте SUBVOX щоразу, коли вони використовуються.

**Як:** Ваші файли cookie шифруються за допомогою Fernet (AES) і зберігаються безпечно. Ви можете скасувати доступ у будь-який час.

### 3. Тримайте токени SUBVOX

Тримайте токени SUBVOX, щоб заробляти частку **пулу власників (20%)**, який розподіляється щодня.

| Рівень | Мінімальний баланс | Вага винагороди | Переваги |
|------|----------------|---------------|----------|
| **Decouverte** | 0 SUBVOX | Без винагород | Безкоштовно для відео < 2 хв, водяний знак, 3/день |
| **Passion** | 500,000 SUBVOX | Вага 1x | Необмежена тривалість, без водяного знаку, експорт SRT |
| **Builder** | 1,000,000 SUBVOX + ключ постачальника | Вага 2x | Винагороди x2, бонус на дохід від вашого ключа постачальника |

**Як це працює:** Кожні 24 години пул власників розподіляється пропорційно. Якщо ви володієте 10% усіх зважених SUBVOX, ви отримуєте 10% пулу цього дня.

## Токен SUBVOX та рівні

### Ціноутворення

Вартість перекладу розраховується за тривалістю відео. Ціни вказані в SUBVOX, за мову.

| Тривалість відео | Ціна (SUBVOX) |
|---------------|----------------|
| < 2 хв | **Безкоштовно** (рівень Decouverte) |
| 2-5 хв | 500 |
| 5-15 хв | 1,500 |
| 15-30 хв | 3,000 |
| 30-60 хв | 6,000 |
| 60-90 хв | 10,000 |

**Знижка на багатомовність:** 20% знижки на кожну додаткову мову.

### Токен

| Властивість | Значення |
|----------|-------|
| Назва токена | SUBVOX |
| Мережа | Solana (devnet для тестування, mainnet TBD) |
| Адреса токена | Незабаром |
| Десяткові знаки | 6 |

### Розподіл доходу за переклад

| Частка | Хто отримує | За що |
|-------|------------|----------|
| **60%** | Постачальники | Розподіляється пропорційно між використаними сервісами: Groq (транскрипція), DeepSeek (переклад), OpenRouter (візуальний аналіз), файли cookie (доступ до платформ) |
| **10%** | Платформа | Хостинг, розробка, операційна діяльність |
| **20%** | Пул власників | Розподіляється щодня серед усіх власників токенів SUBVOX |
| **5%** | Спалюється | Токени назавжди вилучаються з обігу |
| **5%** | Операційні витрати | Інфраструктурні витрати, ретрансляція API, обслуговування |

Розподіл відбувається **автоматично** кожні 24 години через cron-завдання. Жодного ручного втручання.

**Приклад:** Завдання, яке використовує Groq + DeepSeek, ділить 60% порівну (30% + 30%). Завдання, яке використовує всі чотири сервіси, ділить пропорційно (25% + 25% + 5% + 5%).

### Як отримати SUBVOX

| Метод | Опис |
|--------|-------------|
| **Зробіть внесок у код** | Надішліть PR до [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) |
| **Поділіться ключем Groq** | Додайте свій ключ до пулу, заробляйте частку пулу постачальників (60%) |
| **Поділіться файлами cookie** | Розблокуйте обмежені платформи та заробляйте винагороди |
| **Повідомте про помилки** | Валідні звіти про помилки приносять токени |
| **Робота в спільноті** | Переклади, документація, дизайн, модерація |

> **Важливо:** SUBVOX — це комунальний токен спільноти. Він не має гарантованої грошової цінності. Ніколи не платіть гроші за SUBVOX.

## Архітектура

Для розробників, які хочуть заглибитися в код:

```
Frontend (Next.js)  -->  Pipeline API (FastAPI)  -->  Celery Worker
                                                         |
                                                         |--> Download (yt-dlp)
                                                         |--> Transcribe (Groq Whisper)
                                                         |--> Translate (DeepSeek + LLaMA)
                                                         |--> Burn subtitles (FFmpeg)
                                                         |--> Deliver
```

| Репозиторій | Видимість | Призначення |
|-----------|-----------|---------|
| [subvox](https://github.com/Nansoouu/subvox) | Публічний | **Цей репозиторій** -- хаб спільноти, документація, пули |
| [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) | Публічний | Рушій пайплайну (відкритий код) |
