Subvox 是一个**开源的视频字幕引擎**，为社区而生。粘贴视频链接，选择语言，几分钟内获得带字幕的视频。

## 为什么选择 Subvox？

视频翻译影响**数十亿人**。韩语教程、德语会议、阿拉伯语讲座、西班牙语博客。语言不应该成为障碍。

**问题：** 现有工具 (Veed, Kapwing, Descript, Opus Clip) 限制时长和导出，并将你锁定在昂贵的订阅中。

**解决方案：** 一个去中心化的社区，每个人都能受益。

| 谁 | 他们获得什么 |
|-----|------------|
| **重度用户** | 以低价进行无限视频翻译。21种语言。 |
| **Groq 密钥持有者** | 分享未使用的密钥，每次翻译获得55%收益。 |
| **代币持有者** | 持有 500K+ SUBVOX，每日获得奖励。 |
| **贡献者** | 报告错误、编写代码、翻译。赚取 SUBVOX。 |
| **所有人** | 透明可审计的系统。我们保证每个人都能获得报酬。 |

## 流水线如何工作

### 1. 下载
通过 yt-dlp 下载。分享加密的 cookies，每次使用赚取 SUBVOX。

### 2. 转录
通过 Groq Whisper 转录。**社区 Groq 池**提供免费额度。分享密钥赚取55%的费用。

### 3. 翻译
通过 DeepSeek 翻译字幕。

### 4. 烧录
通过 FFmpeg 将字幕烧录到视频中。

### 5. 交付
带字幕的视频已准备就绪，附带可下载的 SRT/VTT 文件。

## 如何赚钱

**1. 分享你的 Groq 密钥** -- 每次使用你的密钥翻译赚取55%。

**2. 分享你的 cookies** -- 解锁受限平台并赚取 SUBVOX。

**3. 持有 SUBVOX** -- 持有者获得10%的费用，每日分配。
