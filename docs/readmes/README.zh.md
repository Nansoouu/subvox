## 为什么选择 Subvox？

视频翻译影响着**数十亿人**。韩语教程、德语会议、阿拉伯语讲座、西班牙语博客。语言不应成为障碍。

**问题所在：** 现有工具（Veed、Kapwing、Descript、Opus Clip）限制你只能处理几分钟的内容，限制导出，并将你锁定在昂贵的订阅中。它们对理应可及的服务收取高昂费用。

**解决方案：** 一个去中心化的社区，让每个人都能受益。

| 谁 | 他们能得到什么 |
|-----|---------------|
| **重度用户** | 以竞争对手价格的一小部分翻译无限量的视频。无时间限制。支持21种语言。 |
| **Groq 密钥持有者** | 分享你未使用的密钥，并从60%的提供者池中赚取分成。被动收入。 |
| **代币持有者** | 质押50万+ SUBVOX，从20%的持有者池中赚取每日奖励。 |
| **Bug 修复者与贡献者** | 报告错误、编写代码、翻译语言。赚取具有实际货币价值的 SUBVOX。 |
| **所有人** | 透明、可审计、链上操作。我们保证每个人都能获得报酬，系统正常运行。 |

无订阅。无隐藏费用。无企业议程。由社区构建，为社区服务。

## 流水线如何工作

每次翻译都经过5个步骤。以下是**社区**如何为每一步提供动力。

> 💡 **想了解幕后工作原理？** 流水线完全开源，位于 [github.com/Nansoouu/subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline)。你可以自行运行、检查每一行代码并贡献。

### 1. 下载
你的视频 URL 通过 yt-dlp 下载。社区成员可以分享加密的 cookies 以解锁受限平台（YouTube 等）。

**你赚取：** 分享你的 cookies，每次被使用时获得奖励。

### 2. 转录
语音通过 Groq Whisper 转录为文本。**社区 Groq 池**提供免费转录额度。

**你赚取：** 将你的 Groq API 密钥分享到池中。提供者份额（翻译成本的60%，在所有使用的服务中分配）归密钥提供者所有。

### 3. 翻译
字幕通过 DeepSeek 翻译成目标语言（计划中支持本地 LLaMA 作为后备）。

### 4. 烧录
字幕通过 FFmpeg 烧录到视频中。计划为长视频提供 GPU 支持。

### 5. 交付
你的带字幕视频已准备就绪，附带可下载的 SRT/VTT 文件。

## 赚钱

Subvox 的设计使得**任何人都能通过贡献生态系统赚钱**。无需编码。

### 1. 分享你的 Groq 密钥

将你的 Groq API 密钥添加到社区池中，每次你的密钥被用于转录时，赚取**60% 提供者池**的分成。

| 你的密钥容量 | 你赚取什么 |
|--------------------|---------------|
| 50% 个人 / 50% 共享 | 一半密钥留给自己，一半给池子（两者你都能赚取） |
| 30 分钟每日限制 | 每天最多30分钟留给自己，其余共享 |
| 无限 | 完全个人访问 + 共享部分的全部池奖励 |

**如何操作：** 在 Web 应用上连接你的钱包（即将推出），前往 计费 > Groq 池，粘贴你的密钥，设置你的个人比例。

### 2. 分享你的 Cookies

某些平台（YouTube 等）需要身份验证 cookies 才能下载视频。分享你的加密 cookies，每次被使用时赚取 SUBVOX。

**如何操作：** 你的 cookies 使用 Fernet（AES）加密并安全存储。你可以随时撤销访问权限。

### 3. 持有 SUBVOX 代币

持有 SUBVOX 代币，从**20% 持有者池**中赚取分成，每日分配。

| 等级 | 最低余额 | 奖励权重 | 权益 |
|------|----------------|---------------|----------|
| **发现者** | 0 SUBVOX | 无奖励 | 免费处理 < 2 分钟的视频，带水印，每天3次 |
| **爱好者** | 500,000 SUBVOX | 1x 权重 | 无限时长，无水印，SRT 导出 |
| **建设者** | 1,000,000 SUBVOX + 提供者密钥 | 2x 权重 | 2倍奖励，提供者密钥收入加成 |

**工作原理：** 每24小时，持有者池按比例分配。如果你持有所有加权 SUBVOX 的10%，你将获得当天池子的10%。

## SUBVOX 代币与等级

### 定价

翻译成本按视频时长计算。价格以 SUBVOX 计价，按语言计费。

| 视频时长 | 价格 (SUBVOX) |
|---------------|----------------|
| < 2 分钟 | **免费**（发现者等级） |
| 2-5 分钟 | 500 |
| 5-15 分钟 | 1,500 |
| 15-30 分钟 | 3,000 |
| 30-60 分钟 | 6,000 |
| 60-90 分钟 | 10,000 |

**多语言折扣：** 每增加一种语言，额外享受20%折扣。

### 代币

| 属性 | 值 |
|----------|-------|
| 代币名称 | SUBVOX |
| 网络 | Solana（devnet 用于测试，主网待定） |
| 代币地址 | 即将推出 |
| 小数位数 | 6 |

### 每次翻译的收入分配

| 份额 | 谁获得 | 用途 |
|-------|------------|----------|
| **60%** | 提供者 | 按使用的服务比例分配：Groq（转录）、DeepSeek（翻译）、OpenRouter（视觉）、cookies（平台访问） |
| **10%** | 平台 | 托管、开发、运营 |
| **20%** | 持有者池 | 每日分配给所有 SUBVOX 代币持有者 |
| **5%** | 销毁 | 代币永久退出流通 |
| **5%** | 运营费用 | 基础设施成本、API 中继、维护 |

分配通过 cron 作业**自动**每24小时进行。无需人工干预。

**示例：** 使用 Groq + DeepSeek 的任务将60%平均分配（30% + 30%）。使用全部四项服务的任务按比例分配（25% + 25% + 5% + 5%）。

### 如何获取 SUBVOX

| 方法 | 描述 |
|--------|-------------|
| **贡献代码** | 向 [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) 提交 PR |
| **分享 Groq 密钥** | 将你的密钥添加到池中，赚取60%提供者池的分成 |
| **分享 Cookies** | 解锁受限平台并赚取奖励 |
| **报告 Bug** | 有效的 Bug 报告可赚取代币 |
| **社区工作** | 翻译、文档、设计、管理 |

> **重要提示：** SUBVOX 是一种社区实用代币。它没有保证的货币价值。切勿为 SUBVOX 支付金钱。

## 架构

对于想要深入代码的开发者：

```
前端 (Next.js)  -->  流水线 API (FastAPI)  -->  Celery Worker
                                                         |
                                                         |--> 下载 (yt-dlp)
                                                         |--> 转录 (Groq Whisper)
                                                         |--> 翻译 (DeepSeek + LLaMA)
                                                         |--> 烧录字幕 (FFmpeg)
                                                         |--> 交付
```

| 仓库 | 可见性 | 用途 |
|-----------|-----------|---------|
| [subvox](https://github.com/Nansoouu/subvox) | 公开 | **本仓库** -- 社区中心、文档、池子 |
| [subvox-pipeline](https://github.com/Nansoouu/subvox-pipeline) | 公开 | 流水线引擎（开源） |
